## **图表**

封装代表单个 Excel 图表的对象。

|属性名称|财产种类|可空|只读|默认值|描述|
|:- |:- |:- |:- |:- |:- |
|自动扩展|布尔值|真的|错误的||如果 Microsoft Excel 缩放三维图表，使其大小更接近等效二维图表，则为 True。RightAngleAxes 属性必须为 True。|
|后墙|类别：墙壁|真的|错误的||返回代表三维图表后墙的对象。|
|分类轴|类别：轴|真的|错误的||获取图表的 X 轴。|
|图表区域|类：ChartArea|真的|错误的||获取工作表中的图表区域。|
|图表数据表|类：ChartDataTable|真的|错误的||代表图表数据表。|
|图表对象|类：LinkElement|真的|错误的||表示图表形状；|
|深度百分比|整数|真的|错误的||以图表宽度的百分比 (20% 到 2000% 之间) 表示三维图表的深度。|
|海拔|整数|真的|错误的||表示三维图表视图的高度（以度为单位）。|
|首片角度|整数|真的|错误的||获取或设置第一个饼图或圆环图切片的角度（以度为单位）（从垂直方向顺时针旋转）。仅适用于饼图、三维饼图和圆环图，范围为 0 到 360。|
|地面|类别：地板|真的|错误的||返回代表三维图表墙的对象。|
|间隙深度|整数|真的|错误的||获取或设置三维图表中数据系列之间的距离（以标记宽度的百分比表示）。此属性的值必须介于 0 和 500 之间。|
|间隙宽度|整数|真的|错误的||返回或设置条形或列簇之间的间距，以条形或列宽度的百分比表示。此属性的值必须介于 0 到 500 之间。|
|高度百分比|整数|真的|错误的||返回或设置三维图表的高度占图表宽度的百分比（5% 到 500% 之间）。|
|隐藏数据透视字段按钮|布尔值|真的|错误的||指示仅当图表为数据透视图时是否隐藏数据透视图字段按钮。|
|是3D|布尔值|真的|错误的||指示图表是否为三维图表。|
|有角矩形|布尔值|真的|错误的||获取或设置一个值，指示图表区域是否为矩形角。默认为 true。|
|传奇|类别：传奇|真的|错误的||获取图表图例。|
|姓名|细绳|真的|错误的||代表图表名称。|
| N系列|类别：系列商品|真的|错误的||获取表示图表中数据系列的集合。|
|页面设置|类：LinkElement|真的|错误的||代表此图表中的页面设置描述。|
|看法|整数|真的|错误的||返回或设置三维图表视图的透视图。必须介于 0 和 100 之间。如果 RightAngleAxes 属性为 True，则忽略此属性。|
|枢轴来源|细绳|真的|错误的||来源为pivotTable的数据，若PivotSource不为空，则图表为PivotChart。|
|放置|细绳|真的|错误的||表示图表附加到其下方单元格的方式。|
|绘图区|类：PlotArea|真的|错误的||获取包含轴刻度标签的图表绘图区域。|
| PlotEmptyCells类型|细绳|真的|错误的||获取并设置如何绘制空单元格。|
|绘制可见单元格|布尔值|真的|错误的||指示是否仅绘制可见单元格。|
|打印尺寸|细绳|真的|错误的||获取并设置打印图表的尺寸。|
|直角轴|布尔值|真的|错误的||如果图表轴成直角，则为 True。仅适用于三维图表（除 3D 柱形图和三维饼图外）。|
|旋转角度|整数|真的|错误的||表示三维图表视图的旋转（绘图区绕 z 轴的旋转，以度为单位）。|
|第二分类轴|类：LinkElement|真的|错误的||获取图表的第二个 X 轴。|
|第二值轴|类：LinkElement|真的|错误的||获取图表的第二个 Y 轴。|
|系列轴|类：LinkElement|真的|错误的||获取图表的系列轴。|
|形状|类：LinkElement|真的|错误的||返回此图表中的所有绘图形状。|
|显示数据表|布尔值|真的|错误的||获取或设置一个值，指示图表是否显示数据表。|
|显示图例|布尔值|真的|错误的||获取或设置一个值，指示是否显示图表图例。默认为 true。|
|侧壁|类：LinkElement|真的|错误的||返回代表三维图表侧壁的对象。|
|窗口大小|布尔值|真的|错误的||如果 Microsoft Excel 调整图表大小以匹配图表工作表窗口的大小，则为真。|
|风格|整数|真的|错误的||获取并设置内置样式。|
|标题|类：LinkElement|真的|错误的||代表图表标题。|
|类型|细绳|真的|错误的||表示图表类型。|
|价值轴|类别：轴|真的|错误的||获取图表的 Y 轴。|
|墙壁|类：LinkElement|真的|错误的||返回代表三维图表墙的对象。|
|WallsAndGridlines2D|布尔值|真的|错误的||如果在三维图表上以二维方式绘制网格线，则该值等于 True。|
|关联|类别：链接|真的|错误的|||
