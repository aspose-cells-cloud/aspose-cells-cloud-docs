# 最重要的发布

- **Aspose.Cells Cloud 4.0 最新版本现已发布。此版本经过重新设计，更加精简高效，确保稳定可靠的使用体验。**
- **最新的SDK整合了Aspose.Cells Cloud 3.0和4.0的所有API，简化了开发流程，使其更加实用和便捷。**
