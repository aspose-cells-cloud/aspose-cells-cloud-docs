此 REST API 表示 `protect` Excel 文件。
## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/protect
 
```
请求参数为：
 
|参数名称|类型|路径/查询字符串/HTTPBody|描述|
|:- |:- |:- |:- |
|文件|文件|表单数据|要上传的文件|
|保护工作簿请求|班级|表单数据|保护工作簿请求|
 
这[开放API规范](https://reference.aspose.cloud/cells/#/LightCells/PostProtect)定义了一个可公开访问的编程接口，并允许您直接从 Web 浏览器执行 REST 交互。
 
您可以使用cURL命令行工具轻松访问Aspose.Cells Web服务。以下示例展示如何使用 cURL 呼叫云端 API。

{{< tab tabNum="1" >}}
 
```bash
 
curl -v "http://api.aspose.cloud/v3.0/cells/protect" \
-X POST \
-H "Content-Type: application/json" \
-H "Accept: application/json" \
-H "Authorization: Bearer <jwt token>" \
-F 'xxxxx1=@xxxx1.xlsx' \
-F 'xxxxx2=@xxxx2.xlsx' \
-F 'protectWorkbookRequest={AwaysOpenReadOnly:true,EncryptWithPassword:"123456"} 
```
