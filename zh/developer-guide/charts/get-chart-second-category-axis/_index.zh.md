此 REST API 表示获取图表第二类轴。
 
## RSET API
 
```bash
 
GET http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/charts/{chartIndex}/secondcategoryaxis
 
```
请求参数为：
 
|参数名称|类型|路径/查询字符串/HTTPBody|描述|
|:- |:- |:- |:- |
|姓名|细绳|小路||
|工作表名称|细绳|小路||
|图表索引|整数|小路||
|文件夹|细绳|询问||
|存储名称|细绳|询问||
 
 tget-图表-第二类-轴复制[开放API规范](https://apireference.aspose.cloud/cells/#/Charts/GetChartSecondCategoryAxis)定义了一个可公开访问的编程接口，并允许您直接从 Web 浏览器执行 REST 交互。
