此 REST API 表示更新图表属性
 
## 重置 API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/charts/{chartIndex}
 
```
请求参数为：
 
|参数名称|类型|路径/查询字符串/HTTPBody|描述|
|:- |:- |:- |:- |
|姓名|细绳|小路||
|工作表名称|细绳|小路||
|图表索引|整数|小路||
|图表||身体||
|文件夹|细绳|询问||
|存储名称|细绳|询问|存储名称。|

<br/>
 
这[OpenAPI 规范](https://apireference.aspose.cloud/cells/#/Charts/PostWorksheetChart)定义一个可公开访问的编程接口，并允许您直接从 Web 浏览器执行 REST 交互。
 
您可以使用 cURL 命令行工具轻松访问 Aspose.Cells Web 服务。以下示例显示如何使用 cURL 调用云 API。
