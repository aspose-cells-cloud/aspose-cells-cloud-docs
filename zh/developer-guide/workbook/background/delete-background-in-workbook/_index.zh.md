此 REST API 指示删除 Excel 工作簿上的背景。

**查询参数**

|参数名称|类型|描述|
|:- |:- |:- |
|文件夹|细绳|原始工作簿文件夹。|
|存储名称|细绳|存储名称。|


## 休息 API

|**API**|**类型**|**描述**|**资源链接**|
|:- |:- |:- |:- |
|/细胞/{名称}/背景|删除|删除excel文件中的背景|[删除工作簿背景](https://apireference.aspose.cloud/cells/#/Workbook/DeleteWorkbookBackground)|

这[OpenAPI 规范](https://apireference.aspose.cloud/cells/#/Workbook/DeleteDecryptDDeleteWorkbookBackgroundocument)定义了一个可公开访问的编程接口，并允许您直接从 Web 浏览器执行 REST 交互。

您可以使用**cURL**用于轻松访问 Aspose.Cells Web 服务的命令行工具。以下示例显示如何使用 cURL 调用 Cloud API。


{{< tabs tabTotal="2" tabID="1" tabName1="Request" tabName2="Response" >}}

{{< tab tabNum="1" >}}

```java

curl -X DELETE "https://api.aspose.cloud/v3.0/cells/Book1.xlsx/background?folder=DotnetFiles" -H "accept: application/json" -H "x-aspose-client: Containerize.Swagger"

```
