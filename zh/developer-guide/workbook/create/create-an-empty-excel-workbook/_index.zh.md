这个 REST API 表示创建一个 `empty workbook`。

**查询参数**

|参数名称|类型|描述|
|:- |:- |:- |
|模板文件|细绳||
|数据文件|细绳||
|是WriteOver|细绳|真假|
|文件夹|细绳|原始工作簿文件夹。|
|存储名称|细绳|存储名称。|

**请求正文参数**

|参数名称|类型|描述|
|:- |:- |:- |
|数据|文件||


## 休息 API

|**API**|**类型**|**描述**|**资源链接**|
|:- |:- |:- |:- |
|/细胞/{名称}|放|创建一个空工作簿|[将工作簿创建](https://apireference.aspose.cloud/cells/#/Workbook/PutWorkbookCreate)|


这[OpenAPI 规范](https://apireference.aspose.cloud/cells/#/Workbook/PutWorkbookCreate)定义了一个可公开访问的编程接口，并允许您直接从 Web 浏览器执行 REST 交互。

您可以使用**cURL**用于轻松访问 Aspose.Cells Web 服务的命令行工具。以下示例显示如何使用 cURL 调用 Cloud API。

```java

curl -X PUT "https://api.aspose.cloud/v3.0/cells/newworkbook.xlsx?isWriteOver=false" -H "accept: application/json" -H "x-aspose-client: Containerize.Swagger"

```
