Этот REST API указывает на `split` мульти Excel файлов.

## РСЕТ API

```bash
 
POST http://api.aspose.cloud/v3.0/cells/split
 
```
 Параметры запроса:
 
| Имя параметра| Тип| Путь/строка запроса/HTTPBody|Описание|
|:- |:- |:- |:- |
| файл| файл| formData| Файл для загрузки|
| формат| нить| запрос||
| пароль| нить| запрос||
| от| целое число| запрос||
| к| целое число| запрос||
 
[Спецификация OpenAPI](https://apireference.aspose.cloud/cells/#/LiteCells/PostSplit) определяет общедоступный программный интерфейс и позволяет выполнять REST-взаимодействия непосредственно из веб-браузера.
 
Вы можете использовать инструмент командной строки cURL для простого доступа к веб-службам Aspose.Cells. В следующем примере показано, как звонить в Cloud API с номером cURL.
