Этот REST API указывает на `get files`.
 
## РСЕТ API
 
```bash
 
GET http://api.aspose.cloud/v3.0/cells/storage/folder/{path}
 
```
 Параметры запроса:
 
| Имя параметра| Тип| Путь/строка запроса/HTTPBody|Описание|
|:- |:- |:- |:- |
| путь| нить| путь| Путь к папке, например '/folder'|
| имя_хранилища| нить| запрос| Имя хранилища|
 
[Спецификация OpenAPI](https://apireference.aspose.cloud/cells/#/Folder/GetFilesList) определяет общедоступный программный интерфейс и позволяет выполнять REST-взаимодействия непосредственно из веб-браузера.
 
Вы можете использовать инструмент командной строки cURL для простого доступа к веб-службам Aspose.Cells. В следующем примере показано, как звонить в Cloud API с номером cURL.
