Этот REST API указывает на удаление `metadata` из нескольких файлов Excel.


## РСЕТ API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/metadata/delete
 
```
 Параметры запроса:
 
| Имя параметра| Тип| Путь/строка запроса/HTTPBody|Описание|
|:- |:- |:- |:- |
| файл| файл| formData| Файл для загрузки|
| тип| нить| запрос| все|
 
[Спецификация OpenAPI](https://apireference.aspose.cloud/cells/#/DeleteMetadata) определяет общедоступный программный интерфейс и позволяет выполнять REST-взаимодействия непосредственно из веб-браузера.
 
Вы можете использовать инструмент командной строки cURL для простого доступа к веб-службам Aspose.Cells. В следующем примере показано, как звонить в Cloud API с номером cURL.
