Этот REST API указывает на обновление `metadata` из нескольких файлов Excel.

## РСЕТ API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/metadata/update
 
```
 Параметры запроса:
 
| Имя параметра| Тип| Путь/строка запроса/HTTPBody|Описание|
|:- |:- |:- |:- |
| файл| файл| данные формы| Файл для загрузки|
| Свойства документа|| тело| Cells Документ собственность.|
 
[Спецификация OpenAPI](https://apireference.aspose.cloud/cells/#/PostMetadata) определяет общедоступный интерфейс программирования и позволяет выполнять взаимодействие с REST непосредственно из веб-браузера.
 
Вы можете использовать инструмент командной строки cURL для легкого доступа к веб-службам Aspose.Cells. В следующем примере показано, как позвонить на Cloud API с помощью cURL.

```bash
 
curl -v "http://api.aspose.cloud/v3.0/cells/metadata/update" \
-X POST \
-H "Content-Type: application/json" \
-H "Accept: application/json" \
-H "Authorization: Bearer <jwt token>" \
-F 'xxxxx1=@xxxx1.xlsx' \
-F 'xxxxx2=@xxxx2.xlsx'  \
-d '[{name="test",value="test"}]'
 
```
