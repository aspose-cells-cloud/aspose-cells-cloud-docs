Этот REST API указывает на удаление фильтра даты на листе Excel.
            
 
## РСЕТ API
 
```bash
 
DELETE http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/autoFilter/dateFilter
 
```
 Параметры запроса:
 
| Имя параметра| Тип| Путь/строка запроса/HTTPBody|Описание|
|:- |:- |:- |:- |
| имя| нить| путь||
| имя листа| нить| путь||
| полеиндекс| целое число| запрос||
| тип даты и времени группировки| нить| запрос||
| год| целое число| запрос|0 |
| месяц| целое число| запрос|0 |
| день| целое число| запрос|0 |
| час| целое число| запрос|0 |
| минута| целое число| запрос|0 |
| второй| целое число| запрос|0 |
| папка| нить| запрос||
| имя_хранилища| нить| запрос| имя хранилища.|
 
[Спецификация OpenAPI](https://apireference.aspose.cloud/cells/#/AutoFilter/DeleteWorksheetDateFilter) определяет общедоступный интерфейс программирования и позволяет выполнять взаимодействие с REST непосредственно из веб-браузера.
 
Вы можете использовать инструмент командной строки cURL для легкого доступа к веб-службам Aspose.Cells. В следующем примере показано, как позвонить на Cloud API с помощью cURL.
