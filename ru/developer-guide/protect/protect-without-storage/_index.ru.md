Этот REST API указывает на файлы `protect` Excel.
## РСЕТ API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/protect
 
```
 Параметры запроса:
 
| Имя параметра| Тип| Путь/строка запроса/HTTPBody|Описание|
|:- |:- |:- |:- |
| файл| файл| данные формы| Файл для загрузки|
| ProtectWorkbookRequest| сорт| данные формы| Защитить запрос книги|
 
[Спецификация OpenAPI](https://reference.aspose.cloud/cells/#/LightCells/PostProtect) определяет общедоступный интерфейс программирования и позволяет выполнять взаимодействие с REST непосредственно из веб-браузера.
 
Вы можете использовать инструмент командной строки cURL для легкого доступа к веб-службам Aspose.Cells. В следующем примере показано, как позвонить на Cloud API с помощью cURL.

```bash
 
curl -v "http://api.aspose.cloud/v3.0/cells/protect" \
-X POST \
-H "Content-Type: application/json" \
-H "Accept: application/json" \
-H "Authorization: Bearer <jwt token>" \
-F 'xxxxx1=@xxxx1.xlsx' \
-F 'xxxxx2=@xxxx2.xlsx' \
-F 'protectWorkbookRequest={AwaysOpenReadOnly:true,EncryptWithPassword:"123456"} 
```
