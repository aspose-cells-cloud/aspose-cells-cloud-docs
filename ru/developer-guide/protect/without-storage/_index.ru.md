Этот REST API указывает на файлы `protect` Excel.

## РСЕТ API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/protect
 
```
 Параметры запроса:
 
| Имя параметра| Тип| Путь/строка запроса/HTTPBody|Описание|
|:- |:- |:- |:- |
| файл| файл| данные формы| Файл для загрузки|
| пароль| нить| запрос||
 
[Спецификация OpenAPI](https://apireference.aspose.cloud/cells/#/LightCells/PostProtect) определяет общедоступный интерфейс программирования и позволяет выполнять взаимодействие с REST непосредственно из веб-браузера.
 
Вы можете использовать инструмент командной строки cURL для легкого доступа к веб-службам Aspose.Cells. В следующем примере показано, как позвонить на Cloud API с помощью cURL.

curl -v "http://api.aspose.cloud/v3.0/cells/protect?password=123456" \

-H "Accept: application/json" \
-H "Authorization: Bearer <jwt token>" \
