## **DataColumnFillValue**

 Означает, что столбец данных заполнен указанным значением.

| Имя свойства| Тип недвижимости| Обнуляемый| Только чтение| Значение по умолчанию| Описание|
|:- |:- |:- |:- |:- |:- |
| Индекс Столбца| Целое число| Истинный| ЛОЖЬ|| Представляет указанный индекс столбца.|
| DataFillValue| Класс:DataFillValue| Истинный| ЛОЖЬ|| Представляет указанное значение заполнения данных.|
