## **ooxmlSaveOptions**

 Представляет параметры сохранения файла ooxml.

| Имя свойства| Тип недвижимости| Обнуляемый| Только чтение| Значение по умолчанию| Описание|
|:- |:- |:- |:- |:- |:- |
| ЭкспортЯчейкаИмя| логическое значение| Истинный| ЛОЖЬ|| Указывает, экспортируется ли имя ячейки в файл Excel2007 .xlsx (.xlsm, .xltx, .xltm). Если к выходному файлу может получить доступ SQL Server DTS, это значение должно быть истинным. Установка значения false значительно повысит производительность и уменьшит размер файла при создании большого файла. Значение по умолчанию — ложь.|
| ОбновитьZoom| логическое значение| Истинный| ЛОЖЬ|| Указывает, обновляется ли коэффициент масштабирования перед сохранением файла, если свойства PageSetup.FitToPagesWide и PageSetup.FitToPagesTall управляют масштабированием листа.|
| ВключитьZip64| логическое значение| Истинный| ЛОЖЬ||Всегда используйте расширения ZIP64 при создании zip-архивов, даже если в этом нет необходимости.|
| ВстроитьOoxmlAsOleObject| логическое значение| Истинный| ЛОЖЬ|| Указывает, встраиваются ли файлы Ooxml OleObject в качестве объекта ole.|
| Тип сжатия| Нить| Истинный| ЛОЖЬ|| Получает и задает тип сжатия для файла ooxml.|
| СохранитьФормат| Нить| Истинный| ЛОЖЬ|||
| КэшированныйФайлПапка| Нить| Истинный| ЛОЖЬ|||
| Очистить данные| логическое значение| Истинный| ЛОЖЬ|||
| Создать каталог| логическое значение| Истинный| ЛОЖЬ|||
| ВключитьHTTP-сжатие| логическое значение| Истинный| ЛОЖЬ|||
| ОбновитьДиаграммуКэш| логическое значение| Истинный| ЛОЖЬ|||
| Сортировать имена| логическое значение| Истинный| ЛОЖЬ|||
