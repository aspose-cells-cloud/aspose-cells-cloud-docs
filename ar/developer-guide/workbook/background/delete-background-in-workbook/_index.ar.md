## REST API

|**API**|**يكتب**|**وصف**|**رابط الموارد**|
|:- |:- |:- |:- |
|/ الخلايا / {name} / الخلفية|يمسح|حذف الخلفية في ملف اكسل|[DeleteWorkbookBackground](https://apireference.aspose.cloud/cells/#/Workbook/DeleteWorkbookBackground)|

 ال[مواصفات OpenAPI](https://apireference.aspose.cloud/cells/#/Workbook/DeleteDecryptDDeleteWorkbookBackgroundocument) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويتيح لك إجراء تفاعلات REST مباشرة من مستعرض ويب.

 يمكنك استخدام**cURL** أداة سطر الأوامر للوصول إلى خدمات الويب Aspose.Cells بسهولة. يوضح المثال التالي كيفية إجراء مكالمات إلى Cloud API مع cURL.
