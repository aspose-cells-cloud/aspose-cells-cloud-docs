يشير REST API إلى `delete folder`.
 
## رسيت API
 
```bash
 
DELETE http://api.aspose.cloud/v3.0/cells/storage/folder/{path}
 
```
 معلمات الطلب هي:
 
| اسم المعلمة| يكتب| المسار/سلسلة الاستعلام/HTTPBody|وصف|
|:- |:- |:- |:- |
| طريق| خيط| طريق| مسار المجلد، على سبيل المثال "/مجلد"|
| اسم التخزين| خيط| استفسار| اسم التخزين|
| العودية| منطقية| استفسار| خطأ شنيع|

 
 ال[مواصفات OpenAPI](https://apireference.aspose.cloud/cells/#/Folder/DeleteFolder) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويتيح لك تنفيذ تفاعلات REST مباشرة من متصفح الويب.
 
يمكنك استخدام أداة سطر الأوامر cURL للوصول إلى خدمات الويب Aspose.Cells بسهولة. يوضح المثال التالي كيفية إجراء مكالمات إلى Cloud API مع cURL.
