يشير هذا REST API إلى `get files`.
 
## RSET API
 
```bash
 
GET http://api.aspose.cloud/v3.0/cells/storage/folder/{path}
 
```
 معلمات الطلب هي:
 
| اسم المعلمة| يكتب| المسار / سلسلة الاستعلام / HTTPBody|وصف|
|:- |:- |:- |:- |
| طريق| خيط| طريق| مسار المجلد مثل "/ folder"|
| اسم التخزين| خيط| استفسار| اسم التخزين|
 
 ال[مواصفات OpenAPI](https://apireference.aspose.cloud/cells/#/Folder/GetFilesList) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويتيح لك إجراء تفاعلات REST مباشرة من مستعرض ويب.
 
يمكنك استخدام أداة سطر الأوامر cURL للوصول إلى خدمات الويب Aspose.Cells بسهولة. يوضح المثال التالي كيفية إجراء مكالمات إلى Cloud API مع cURL.

-H "Authorization: Bearer <jwt token>"
