يشير REST API إلى الحصول على `file versions`.
 
## رسيت API
 
```bash
 
GET http://api.aspose.cloud/v3.0/cells/storage/version/{path}
 
```
 معلمات الطلب هي:
 
| اسم المعلمة| يكتب| المسار/سلسلة الاستعلام/HTTPBody|وصف|
|:- |:- |:- |:- |
| طريق| خيط| طريق| مسار الملف على سبيل المثال "/file.ext"|
| اسم التخزين| خيط| استفسار| اسم التخزين|

 
 ال[مواصفات OpenAPI](https://apireference.aspose.cloud/cells/#/Storage/GetFileVersions) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويتيح لك تنفيذ تفاعلات REST مباشرة من متصفح الويب.
 
يمكنك استخدام أداة سطر الأوامر cURL للوصول إلى خدمات الويب Aspose.Cells بسهولة. يوضح المثال التالي كيفية إجراء مكالمات إلى Cloud API مع cURL.
