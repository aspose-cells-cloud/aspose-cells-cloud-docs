## **Excel API: إنشاء مجلد**

```
PUT http://api.aspose.cloud/v4.0/cells/storage/folder/{path}
```

### **وصف الوظيفة**

 ال**إنشاء مجلد** يتيح API للمستخدمين إنشاء مجلد جديد في المسار المحدد ضمن التخزين السحابي لـ Excel API. هذه الوظيفة ضرورية لتنظيم الملفات والحفاظ على دليل منظم.

###  معلمات الطلب**إنشاء مجلد** API هم

| اسم المعلمة| يكتب| المسار/سلسلة الاستعلام/نص HTTP| وصف|
|----------|--|------------------------|---------------------------------|
| طريق| خيط| طريق| المسار الذي سيتم إنشاء المجلد فيه.|
| اسم التخزين| خيط| استفسار| اسم التخزين الذي سيتم استخدامه.|

### **وصف الاستجابة**

```json
{
Void
}
```

## مواصفات OpenAPI

 ال[مواصفات OpenAPI](https://reference.aspose.cloud/cells/#/FolderController/CreateFolder) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويسمح لك بتنفيذ تفاعلات REST مباشرة من متصفح الويب.
