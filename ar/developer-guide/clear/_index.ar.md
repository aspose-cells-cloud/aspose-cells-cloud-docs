يشير هذا REST API إلى `clear` كائن Excel في ملفات Excel.

## RSET API

```bash

POST https://api.aspose.cloud/v3.0/cells/clearobjects

```
 طلب المعلمات هي:
 
| اسم المعلمة| يكتب| المسار / سلسلة الاستعلام / HTTPBody|وصف|
|:- |:- |:- |:- |
| ملف| ملف| بيانات النموذج| ملف للتحميل|
| نوع الكائن| خيط| استفسار| تكرار / أعمدة فارغة / صفوف فارغة / صيغة / محتوى / نمط / رسم بياني / تعليق / صورة / شكل / كائن قائمة / ارتباط تشعبي / oleobject / محوري / التحقق من الصحة / الخلفية|
 
 ال[مواصفات OpenAPI](https://apireference.aspose.cloud/cells/#/LiteCells/PostClearObjects) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويتيح لك إجراء تفاعلات REST مباشرة من مستعرض ويب.
 
يمكنك استخدام أداة سطر الأوامر cURL للوصول إلى خدمات الويب Aspose.Cells بسهولة. يوضح المثال التالي كيفية إجراء مكالمات إلى Cloud API مع cURL.

```bash
 
curl -v "http://api.aspose.cloud/v3.0/cells/clearobjects?objecttype=comment" \
-X POST \
-H "Content-Type: application/json" \
-H "Accept: application/json" \
-H "Authorization: Bearer <jwt token>" \
-F 'xxxxx1=@xxxx1.xlsx' \
-F 'xxxxx2=@xxxx2.xlsx' 
```
