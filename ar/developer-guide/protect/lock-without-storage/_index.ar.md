يشير هذا REST API إلى `lock` Excel الملفات.
## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/lock
 
```
 معلمات الطلب هي:
 
| اسم المعلمة| يكتب| المسار / سلسلة الاستعلام / HTTPBody|وصف|
|:- |:- |:- |:- |
| ملف| ملف| بيانات النموذج| ملف للتحميل|
| كلمة المرور| خيط| استفسار||
 
 ال[مواصفات OpenAPI](https://apireference.aspose.cloud/cells/#/LightCells/PostLock) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويتيح لك إجراء تفاعلات REST مباشرة من مستعرض ويب.
 
يمكنك استخدام أداة سطر الأوامر cURL للوصول إلى خدمات الويب Aspose.Cells بسهولة. يوضح المثال التالي كيفية إجراء مكالمات إلى Cloud API مع cURL.
