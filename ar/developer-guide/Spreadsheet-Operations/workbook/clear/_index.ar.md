يشير هذا REST API إلى كائن Excel `clear` في ملفات Excel.

## RSET API

```bash

POST https://api.aspose.cloud/v3.0/cells/clearobjects

```

معلمات الطلب هي:

| اسم المعلمة| يكتب| المسار/سلسلة الاستعلام/نص HTTP|وصف|
|:- |:- |:- |:- |
| ملف| ملف| نموذج البيانات| الملف المراد تحميله|
| نوع الكائن| خيط| استفسار| تكرار الصفوف/الأعمدة الفارغة/الصفوف الفارغة/الصيغة/المحتوى/النمط/المخطط/التعليق/الصورة/الشكل/كائن القائمة/الارتباط التشعبي/كائن العنصر/الجدول المحوري/التحقق/الخلفية|

 ال[مواصفات OpenAPI](https://apireference.aspose.cloud/cells/#/LightCells/PostClearObjects) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويسمح لك بتنفيذ تفاعلات REST مباشرة من متصفح الويب.

يمكنك استخدام أداة سطر الأوامر cURL للوصول بسهولة إلى خدمات الويب Aspose.Cells. يوضح المثال التالي كيفية إجراء مكالمات إلى Cloud API باستخدام cURL.

```bash
 
curl -v "http://api.aspose.cloud/v3.0/cells/clearobjects?objecttype=comment" \
-X POST \
-H "Content-Type: application/json" \
-H "Accept: application/json" \
-H "Authorization: Bearer <jwt token>" \
-F 'xxxxx1=@xxxx1.xlsx' \
-F 'xxxxx2=@xxxx2.xlsx' 
```
