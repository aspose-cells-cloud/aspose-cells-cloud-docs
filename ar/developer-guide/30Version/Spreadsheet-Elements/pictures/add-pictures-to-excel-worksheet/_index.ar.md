يشير هذا REST API إلى `add` صورة جديدة لورقة عمل Excel.

## RSET API

```bash
 
PUT http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/pictures
 
```

معلمات الطلب هي:

| اسم المعلمة| يكتب| المسار/سلسلة الاستعلام/نص HTTP|وصف|
|:- |:- |:- |:- |
| اسم| خيط| طريق| اسم المصنف.|
| اسم الورقة| خيط| طريق| اسم العبادة.|
| صورة|| جسم| كائن مصور|
| الصف العلوي الأيسر| عدد صحيح| استفسار|0 |
| العمود الأيسر العلوي| عدد صحيح| استفسار|0 |
| الصف الأيمن السفلي| عدد صحيح| استفسار|0 |
| العمود الأيمن السفلي| عدد صحيح| استفسار|0 |
| مسار الصورة| خيط| استفسار| مسار الصورة، إذا لم يتم توفير بيانات الصورة، فسيتم فحصها في نص الطلب.|
| مجلد| خيط| استفسار| مجلد المصنف.|
| اسم التخزين| خيط| استفسار| اسم التخزين.|

 ال[مواصفات OpenAPI](https://apireference.aspose.cloud/cells/#/Pictures/PutWorksheetAddPicture) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويسمح لك بتنفيذ تفاعلات REST مباشرة من متصفح الويب.

يمكنك استخدام أداة سطر الأوامر cURL للوصول بسهولة إلى خدمات الويب Aspose.Cells. يوضح المثال التالي كيفية إجراء مكالمات إلى Cloud API باستخدام cURL.

```bash
 
curl -v "http://api.aspose.com/v1.1/cells/Sample_Test_Book.xls/worksheets/Sheet6/pictures?picturePath=aspose-cloud.png" \
-X PUT \
-H "Content-Type: application/json" \
-H "Accept: application/json" \
-H "Authorization: Bearer <jwt token>"
 
```
