يشير هذا REST API إلى ملاءمة صف تلقائيًا في ورقة عمل Excel.

## RSET API

```bash
 
POST http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/autofitrow
 
```

معلمات الطلب هي:

| اسم المعلمة| يكتب| المسار/سلسلة الاستعلام/نص HTTP|وصف|
|:- |:- |:- |:- |
| اسم| خيط| طريق| اسم الملف.|
| اسم الورقة| خيط| طريق| اسم ورقة العمل.|
| فهرس الصف| عدد صحيح| استفسار| مؤشر الصف.|
| العمود الأول| عدد صحيح| استفسار| فهرس العمود الأول.|
| العمود الأخير| عدد صحيح| استفسار| فهرس العمود الأخير.|
|[خيارات التجهيز التلقائي](/cells/ar/auto-fitter-options) | هدف| جسم| خيارات التركيب التلقائي.|
| مجلد| خيط| استفسار| المجلد.|
| اسم التخزين| خيط| استفسار| اسم التخزين.|

 ال[مواصفات OpenAPI](https://apireference.aspose.cloud/cells/#/Worksheets/PostAutofitWorksheetRow) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويسمح لك بتنفيذ تفاعلات REST مباشرة من متصفح الويب.

يمكنك استخدام أداة سطر الأوامر cURL للوصول بسهولة إلى خدمات الويب Aspose.Cells. يوضح المثال التالي كيفية إجراء مكالمات إلى Cloud API باستخدام cURL.
