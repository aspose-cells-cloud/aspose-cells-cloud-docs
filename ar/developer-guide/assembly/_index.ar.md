يشير REST API إلى بيانات `assembly` في ملف Excel.

## رسيت API

```bash

POST https://api.aspose.cloud/v3.0/cells/assembly

```

 معلمات الطلب هي:
 
| اسم المعلمة| يكتب| المسار/سلسلة الاستعلام/HTTPBody|وصف|
|:- |:- |:- |:- |
| ملف| ملف| بيانات النموذج| ملف للتحميل|
| مصدر البيانات| خيط| استفسار||
| شكل| خيط| استفسار| XLSX|
 
 ال[مواصفات OpenAPI](https://apireference.aspose.cloud/cells/#/LightCells/PostAssemble) يحدد واجهة برمجة يمكن الوصول إليها بشكل عام ويتيح لك تنفيذ تفاعلات REST مباشرة من متصفح الويب.
 
يمكنك استخدام أداة سطر الأوامر cURL للوصول إلى خدمات الويب Aspose.Cells بسهولة. يوضح المثال التالي كيفية إجراء مكالمات إلى Cloud API مع cURL.

```bash
 
curl -v "http://api.aspose.cloud/v3.0/cells/assembly?datasource=ds&format=pdf" \
-X POST \
-H "Content-Type: multipart/form-data" \
-H "Accept: application/json" \
-H "Authorization: Bearer <jwt token>" \
-F 'xxxxx1=@xxxx1.xlsx' \
-F 'xxxxx2=@xxxx2.xlsx' 
```
