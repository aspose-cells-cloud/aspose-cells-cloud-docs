This REST API indicates delete worksheet chart by index.
 
## RSET API
 
```bash
 
DELETE http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/charts/{chartIndex}
 
```
The request parameters are: 
 
| Parameter Name | Type | Path/Query String/HTTPBody | Description| 
| :- | :- | :- |:- | 
| name | string | path | Workbook name. |
| sheetName | string | path | Worksheet name. |
| chartIndex | integer | path | The chart index. |
| folder | string | query | The workbook folder. |
| storageName | string | query | storage name. |
 
The [OpenAPI Specification](https://apireference.aspose.cloud/cells/#/Charts/DeleteWorksheetDeleteChart) defines a publicly accessible programming interface and lets you carry out REST interactions directly from a web browser.
