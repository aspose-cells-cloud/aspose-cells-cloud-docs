## **txtSaveOptions**

 Représente les options d'enregistrement du fichier .txt.

| Nom de la propriété| Type de propriété| Nullable| Lecture seulement| Valeur par défaut| Description|
|:- |:- |:- |:- |:- |:- |
| Type de devis| Chaîne| Vrai| FAUX|| Obtient ou définit comment citer les valeurs dans le fichier texte exporté.|
| Séparateur| Chaîne| Vrai| FAUX|| Obtient et définit le délimiteur de caractères du fichier texte.|
| Chaîne de séparation| Chaîne| Vrai| FAUX|| Obtient et définit la valeur d'une chaîne comme séparateur.|
| Toujours cité| Booléen| Vrai| FAUX|| Indique s'il faut toujours ajouter '"' pour chaque champ. Si vrai, toutes les valeurs seront entre guillemets ; si faux, les valeurs ne seront citées qu'en cas de besoin (par exemple, lorsque les valeurs contiennent des caractères spéciaux tels que '"' , '\n' ou caractère séparateur). La valeur par défaut est fausse.|
| EnregistrerFormat| Chaîne| Vrai| FAUX|||
| Dossier de fichiers mis en cache| Chaîne| Vrai| FAUX|||
| Effacer les données| Booléen| Vrai| FAUX|||
| Créer le répertoire| Booléen| Vrai| FAUX|||
| Activer la compression HTTP| Booléen| Vrai| FAUX|||
| Actualiser le cache de graphiques| Booléen| Vrai| FAUX|||
|Trier les noms| Booléen| Vrai| FAUX|||
| ValidateMergedAreas| Booléen| Vrai| FAUX|||
