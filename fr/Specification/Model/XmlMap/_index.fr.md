## **xmlCarte**

 Représente les informations de la carte XML.

| Nom de la propriété| Type de propriété| Nullable| Lecture seulement| Valeur par défaut| Description|
|:- |:- |:- |:- |:- |:- |
| Nom| Chaîne| Vrai| FAUX|| Renvoie ou définit le nom de l'objet.|
| RootElementName| Chaîne| Vrai| FAUX|| Obtient le nom de l'élément racine.|
| Liaison de données| Classe : XmlDataBinding| Vrai| FAUX|| Obtient une de cette carte.|
