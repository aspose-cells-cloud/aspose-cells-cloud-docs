Ce REST API indique d'ajouter `watermark` sur les fichiers Excel.

## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/watermark
 
```
 Les paramètres de la requête sont :
 
| Le nom du paramètre| Taper| Chemin/Chaîne de requête/HTTPBody|Description|
|:- |:- |:- |:- |
| déposer| déposer| Données de formulaire| Fichier à télécharger|
| texte| chaîne| requête||
| couleur| chaîne| requête||
 
 Le[Spécification OpenAPI](https://apireference.aspose.cloud/cells/#/LightCells/PostWatermark) définit une interface de programmation accessible au public et vous permet d'effectuer des interactions REST directement depuis un navigateur Web.
 
Vous pouvez utiliser l'outil de ligne de commande cURL pour accéder facilement aux services Web Aspose.Cells. L'exemple suivant montre comment passer des appels vers le Cloud API avec cURL.
