Ce REST API indique `batch protection` des fichiers éligibles.
 
## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/batch/protect
 
```
 Les paramètres de la requête sont :
 
| Le nom du paramètre| Taper| Chemin/Chaîne de requête/HTTPBody|Description|
|:- |:- |:- |:- |
| batchProtectRequest|| corps||

**Propriétés de BatchProtectRequest**
 
Nom | Tapez | Descriptif | Remarques
------------ | ------------- | ------------- | -------------
 Dossier Source | chaîne | | [facultatif]Condition de correspondance | CorrespondanceConditionRequest | | [facultatif]Type de protection | chaîne | | [facultatif]Mot de passe | chaîne | | [facultatif]OutFolder | chaîne | | [facultatif]**Propriétés MatchConditionRequest**
 
Nom | Tapez | Descriptif | Remarques
------------ | ------------- | ------------- | -------------
 Modèle Regex | chaîne | | [facultatif]Conditions FullMatch | chaîne[]| | [facultatif]Le[Spécification OpenAPI](https://apireference.aspose.cloud/cells/#/PostProtectConvert) définit une interface de programmation accessible au public et vous permet d'effectuer des interactions REST directement depuis un navigateur Web.
