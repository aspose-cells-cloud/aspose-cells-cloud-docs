## **Excel API : Créer un dossier**

```
PUT http://api.aspose.cloud/v4.0/cells/storage/folder/{path}
```

### **Description de la fonction**

 Le**créer un dossier** API permet aux utilisateurs de créer un nouveau dossier dans le chemin spécifié dans le stockage cloud du Excel API. Cette fonctionnalité est essentielle pour organiser les fichiers et maintenir un répertoire structuré.

###  Les paramètres de la requête de**créer un dossier** API sont

| Nom du paramètre| Taper| Chemin/Chaîne de requête/Corps HTTP| Description|
|----------|--|------------------------|---------------------------------|
| chemin| Chaîne| Chemin| Le chemin où le dossier sera créé.|
| nom de stockage| Chaîne| Requête| Le nom du stockage à utiliser.|

### **Description de la réponse**

```json
{
Void
}
```

## Spécification OpenAPI

 Le[Spécification OpenAPI](https://reference.aspose.cloud/cells/#/FolderController/CreateFolder) définit une interface de programmation accessible au public et vous permet d'effectuer des interactions REST directement à partir d'un navigateur Web.
