Ce REST API indique aux fichiers `unlock` Excel.
## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/unlock
 
```
 Les paramètres de la requête sont :
 
| Le nom du paramètre| Taper| Chemin/chaîne de requête/HTTPBody|Description|
|:- |:- |:- |:- |
| déposer| déposer| Données de formulaire| Fichier à uploader|
| mot de passe| chaîne| mettre en doute||
 
 Le[Spécification OpenAPI](https://apireference.aspose.cloud/cells/#/LiteCells/PostUnlock) définit une interface de programmation accessible au public et permet d'effectuer des interactions REST directement depuis un navigateur Web.
 
Vous pouvez utiliser l'outil de ligne de commande cURL pour accéder facilement aux services Web Aspose.Cells. L'exemple suivant montre comment passer des appels vers le Cloud API avec cURL.
