Bu REST API, `repair` Excel dosyalarını gösterir.

- XLS, XLSX, XLSM, XLSB, ODS ve benzerlerini onarın.
- Çoklu dosyaları destekleyin.

Aspose.Cells Bulut Excel Onarım, bozuk Excel dosyalarındaki verileri kurulum gerektirmeden çevrimiçi kurtarır. Bozuk Excel dosyaları, onları açamayacağınız için sorun olabilir. Bozuk Excel dosyalarındaki verileri kurtarmak için Aspose.Cells Bulut Excel Onarım Uygulamasını deneyebilirsiniz.

## RSET API

```bash

POST https://api.aspose.cloud/v3.0/cells/repair

```

 İstek parametreleri şunlardır:
 
| Parametre adı| Tip| Yol/Sorgu Dizesi/HTTPBody|Tanım|
|:- |:- |:- |:- |
| dosya| dosya| form verisi| Yüklenecek dosya|
| biçim| sicim| sorgu| Çıkış formatı, Varsayılan değer boştur, çıkış formatı giriş dosyası formatına eşittir.|
 
[OpenAPI Spesifikasyonu](https://reference.aspose.cloud/cells/#/LightCells/PostRepair) herkese açık bir programlama arayüzü tanımlar ve REST etkileşimlerini doğrudan bir web tarayıcısından gerçekleştirmenize olanak tanır.
 
Aspose.Cells web servislerine kolayca erişmek için cURL komut satırı aracını kullanabilirsiniz. Aşağıdaki örnekte, cURL ile Cloud API'e nasıl çağrı yapılacağı gösterilmektedir.

-X POST \
-H "Content-Type: multipart/form-data" \
-H "Accept: application/json" \
-H "Authorization: Bearer <jwt token>" \
