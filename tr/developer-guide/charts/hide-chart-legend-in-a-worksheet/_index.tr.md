Bu REST API, grafikteki açıklamayı gizle anlamına gelir
 
## RSET API
 
```bash
 
DELETE http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/charts/{chartIndex}/legend
 
```
 İstek parametreleri şunlardır:
 
| Parametre adı| Tip| Yol/Sorgu Dizesi/HTTPBody|Tanım|
|:- |:- |:- |:- |
| isim| sicim| yol| Çalışma kitabı adı.|
| sayfaAdı| sicim| yol| Çalışma sayfası adı.|
| grafikIndex| tamsayı| yol| Grafik indeksi.|
| dosya| sicim| sorgu| Çalışma kitabı klasörü.|
| depolamaAdı| sicim| sorgu| depolama adı.|


<br/>

 
[OpenAPI Spesifikasyonu](https://apireference.aspose.cloud/cells/#/Charts/DeleteWorksheetChartLegend) herkese açık bir programlama arayüzü tanımlar ve REST etkileşimlerini doğrudan bir web tarayıcısından gerçekleştirmenize olanak tanır.
 
Aspose.Cells web servislerine kolayca erişmek için cURL komut satırı aracını kullanabilirsiniz. Aşağıdaki örnekte, cURL ile Cloud API'e nasıl çağrı yapılacağı gösterilmektedir.
