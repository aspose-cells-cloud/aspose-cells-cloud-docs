Denna REST API indikerar att lägga till `watermark` på Excel filer.

## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/watermark
 
```
 Begärans parametrar är:
 
| Parameternamn| Typ| Sökväg/Frågesträng/HTTPBody|Beskrivning|
|:- |:- |:- |:- |
| fil| fil| formData| Fil att ladda upp|
| text| sträng| fråga||
| Färg| sträng| fråga||
 
 De[OpenAPI-specifikation](https://apireference.aspose.cloud/cells/#/LightCells/PostWatermark) definierar ett allmänt tillgängligt programmeringsgränssnitt och låter dig utföra REST-interaktioner direkt från en webbläsare.
 
Du kan använda cURL kommandoradsverktyg för att enkelt komma åt Aspose.Cells webbtjänster. Följande exempel visar hur man ringer till Cloud API med cURL.

curl -v "http://api.aspose.cloud/v3.0/cells/watermark?text=aspose.cells.cloud&color=004433ff" \

-H "Accept: application/json" \
