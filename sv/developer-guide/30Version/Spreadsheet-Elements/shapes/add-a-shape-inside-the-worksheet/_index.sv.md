Denna REST API anger att en form ska läggas till i ett Excel-arbetsblad.

## RSET API

```bash
 
PUT http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/shapes
 
```

Begäranparametrarna är:

| Parameternamn| Typ| Sökväg/Frågesträng/HTTP-kropp|Beskrivning|
|:- |:- |:- |:- |
| namn| sträng| väg| dokumentnamn.|
| arknamn| sträng| väg| arbetsbladets namn.|
| shapeDTO|| kropp||
|ritningstyp| sträng| fråga| formobjekttyp|
| övre vänsterrad| heltal| fråga| Index för övre vänstra raden.|
| övre vänsterkolumn| heltal| fråga| Index i övre vänstra kolumnen.|
| bästa| heltal| fråga| Representerar den vertikala förskjutningen av Spinner från dess vänstra rad, i pixelenhet.|
| vänster| heltal| fråga| Representerar den horisontella förskjutningen av Spinner från dess vänstra kolumn, i pixelenhet.|
| bredd| heltal| fråga| Representerar Spinnerns höjd, i pixelenhet.|
| höjd| heltal| fråga| Representerar bredden på Spinner, i pixelenhet.|
| mapp| sträng| fråga| Dokumentets mapp.|
| lagringsnamn| sträng| fråga| lagringsnamn.|

 De[OpenAPI-specifikation](https://apireference.aspose.cloud/cells/#/Shapes/PutWorksheetShape) definierar ett offentligt tillgängligt programmeringsgränssnitt och låter dig utföra REST-interaktioner direkt från en webbläsare.

Du kan använda kommandoradsverktyget cURL för att enkelt komma åt Aspose.Cells webbtjänster. Följande exempel visar hur man gör anrop till Cloud API med cURL.
