Denna REST API indikerar visa legend i diagrammet
 
## RSET API
 
```bash
 
PUT http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/charts/{chartIndex}/legend
 
```
 Begärans parametrar är:
 
| Parameternamn| Typ| Sökväg/Frågesträng/HTTPBody|Beskrivning|
|:- |:- |:- |:- |
| namn| sträng| väg| Arbetsbokens namn.|
| arknamn| sträng| väg| Arbetsbladsnamn.|
| diagramIndex| heltal| väg| Diagrammets index.|
| mapp| sträng| fråga| Arbetsboksmappen.|
| lagringsnamn| sträng| fråga| lagringsnamn.|

 
 De[OpenAPI-specifikation](https://apireference.aspose.cloud/cells/#/Charts/PutWorksheetChartLegend) definierar ett allmänt tillgängligt programmeringsgränssnitt och låter dig utföra REST-interaktioner direkt från en webbläsare.
 
Du kan använda cURL kommandoradsverktyg för att enkelt komma åt Aspose.Cells webbtjänster. Följande exempel visar hur man ringer till Cloud API med cURL.
