Denna REST API indikerar uppdateringscell `style` för pivottabell.
 
## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/pivottables/{pivotTableIndex}/Format
 
```
 Begärans parametrar är:
 
| Parameternamn| Typ| Sökväg/Frågesträng/HTTPBody|Beskrivning|
|:- |:- |:- |:- |
| namn| sträng| väg| Dokument namn.|
| arknamn| sträng| väg| Kalkylbladets namn.|
| pivotTableIndex| heltal| väg| Pivottabellindex|
| kolumn| heltal| fråga||
| rad| heltal| fråga||
| stil|| kropp| Stil dto i texten i begäran.|
| behöver Beräkna om| booleskt| fråga| Falsk|
| mapp| sträng| fråga| Dokumentets mapp.|
| lagringsnamn| sträng| fråga| lagringsnamn.|
 
 De[OpenAPI-specifikation](https://apireference.aspose.cloud/cells/#/PivotTables/PostPivotTableCellStyle) definierar ett allmänt tillgängligt programmeringsgränssnitt och låter dig utföra REST-interaktioner direkt från en webbläsare.
