Denna REST API indikerar kopiering av kalkylbladsrader.
 
## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/cells/rows/copy
 
```
 Begärans parametrar är:
 
| Parameternamn| Typ| Sökväg/Frågesträng/HTTPBody|Beskrivning|
|:- |:- |:- |:- |
| namn| sträng| väg|Arbetsbokens namn.|
| arknamn| sträng| väg| Kalkylbladets namn.|
| sourceRowIndex| heltal| fråga| Källradsindex|
| destinationRowIndex| heltal| fråga| Destinationsradindex|
| radNumber| heltal| fråga| Det kopierade radnumret|
| arbetsblad| sträng| fråga| arbetsblad|
| mapp| sträng| fråga| Dokumentmappen.|
| lagringsnamn| sträng| fråga| lagringsnamn.|
 
 De[OpenAPI-specifikation](https://apireference.aspose.cloud/cells/#/Cells/PostCopyWorksheetRows) definierar ett allmänt tillgängligt programmeringsgränssnitt och låter dig utföra REST-interaktioner direkt från en webbläsare.
 
Du kan använda cURL kommandoradsverktyg för att enkelt komma åt Aspose.Cells webbtjänster. Följande exempel visar hur man ringer till Cloud API med cURL.

curl -v "https://api.aspose.cloud/v3.0/cells/test.xlsx/worksheets/Sheet1/cells/rows/copy?sourceRowIndex=1&destinationRowIndex=12&rowNumber=10" \
-X POST \
-H "Content-Type: application/json" \
