Denna REST API indikerar att gruppera rader på ett Excel kalkylblad.
 
## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/cells/rows/group
 
```
 Begärans parametrar är:
 
| Parameternamn| Typ| Sökväg/Frågesträng/HTTPBody|Beskrivning|
|:- |:- |:- |:- |
| namn| sträng| väg|Arbetsbokens namn.|
| arknamn| sträng| väg| Kalkylbladets namn.|
| firstIndex| heltal| fråga| Den första radindex som ska användas.|
| lastIndex| heltal| fråga| Det sista radindexet som ska användas.|
| Dölj| booleskt| fråga| rader synligt tillstånd|
| mapp| sträng| fråga| Dokumentmappen.|
| lagringsnamn| sträng| fråga| lagringsnamn.|
 
 De[OpenAPI-specifikation](https://apireference.aspose.cloud/cells/#/Cells/PostGroupWorksheetRows) definierar ett allmänt tillgängligt programmeringsgränssnitt och låter dig utföra REST-interaktioner direkt från en webbläsare.
 
Du kan använda cURL kommandoradsverktyg för att enkelt komma åt Aspose.Cells webbtjänster. Följande exempel visar hur man ringer till Cloud API med cURL.
