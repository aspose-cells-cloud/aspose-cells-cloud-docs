## **txtGuardarOpciones**

 Representa las opciones para guardar el archivo .txt.

| Nombre de la propiedad| tipo de propiedad| Anulable| Solo lectura| Valor por defecto| Descripción|
|:- |:- |:- |:- |:- |:- |
| Tipo de cotización| Cadena| Verdadero| FALSO|| Obtiene o establece cómo citar valores en el archivo de texto exportado.|
| Separador| Cadena| Verdadero| FALSO|| Obtiene y establece el delimitador char del archivo de texto.|
| Cadena separadora| Cadena| Verdadero| FALSO|| Obtiene y establece el valor de una cadena como separador.|
| Siempre citado| Booleano| Verdadero| FALSO|| Indica si se agrega siempre '"' para cada campo. Si es verdadero, se citarán todos los valores; si es falso, los valores solo se citarán cuando sea necesario (por ejemplo, cuando los valores contengan caracteres especiales como '"', '\n' o carácter separador). El valor predeterminado es falso.|
