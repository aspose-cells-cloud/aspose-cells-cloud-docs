Este REST API indica que los programas obtienen un `maxcolumn` en un archivo Excel cuando el parámetro `cellOrMethodName` es `maxcolumn`.

- **cURL Ejemplo**


{{< tabs tabTotal="2" tabID="11" tabName11="Request" tabName12="Response" >}}

{{< tab tabNum="11" >}}

```java

curl -X GET "http://api.aspose.com/v3.0/cells/myWorkbook.xlsx/worksheets/Sheet1/cells/maxcolumn"  -H "Content-Type: application/json" -H "Accept: application/json"

```
