Este REST API indica que muestra obtener un `maxdatacolumn` en un archivo Excel cuando el parámetro `cellOrMethodName` es `maxdatacolumn`.

- **cURL Ejemplo**


{{< tabs tabTotal="2" tabID="11" tabName11="Request" tabName12="Response" >}}

{{< tab tabNum="11" >}}

```java

curl -X GET "http://api.aspose.com/v3.0/cells/myWorkbook.xlsx/worksheets/Sheet1/cells/maxdatacolumn" -H "Content-Type: application/json" -H "Accept: application/json"

```
