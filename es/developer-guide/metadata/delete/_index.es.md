Este REST API indica eliminar `metadata` de varios archivos Excel.


## RESET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/metadata/delete
 
```
 Los parámetros de la solicitud son:
 
| Nombre del parámetro| Tipo| Ruta/Cadena de consulta/Cuerpo HTTP|Descripción|
|:- |:- |:- |:- |
| archivo| archivo| formularioDatos| Subir Archivo|
| tipo| cadena| consulta| todo|
 
 El[Especificación de API abierta](https://apireference.aspose.cloud/cells/#/DeleteMetadata) define una interfaz de programación de acceso público y le permite realizar interacciones REST directamente desde un navegador web.
 
Puede utilizar la herramienta de línea de comandos cURL para acceder fácilmente a los servicios web Aspose.Cells. El siguiente ejemplo muestra cómo realizar llamadas a Cloud API con cURL.
