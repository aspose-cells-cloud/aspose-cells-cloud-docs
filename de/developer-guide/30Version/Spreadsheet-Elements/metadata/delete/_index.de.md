Dieser REST API gibt an, `metadata` aus mehreren Excel-Dateien zu löschen.

## RSET API

```bash
 
POST http://api.aspose.cloud/v3.0/cells/metadata/delete
 
```

Die Anforderungsparameter sind:

| Parametername| Typ| Pfad/Abfragezeichenfolge/HTTPBody|Beschreibung|
|:- |:- |:- |:- |
| Datei| Datei| formData| Hochzuladende Datei|
| Typ| Schnur| Abfrage| alle|

 Der[OpenAPI-Spezifikation](https://apireference.aspose.cloud/cells/#/DeleteMetadata) definiert eine öffentlich zugängliche Programmierschnittstelle und ermöglicht Ihnen die Durchführung von REST-Interaktionen direkt von einem Webbrowser aus.

Mit dem Befehlszeilentool cURL können Sie problemlos auf die Webdienste Aspose.Cells zugreifen. Das folgende Beispiel zeigt, wie Sie mit cURL Aufrufe an Cloud API tätigen.
