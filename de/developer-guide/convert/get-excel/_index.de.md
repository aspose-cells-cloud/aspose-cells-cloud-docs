Dieser REST API zeigt die `get` Excel-Datei in einem anderen Dateiformat an.


**Abfrageparameter**

|Parametername|Typ|Beschreibung|
|:- |:- |:- |
|Format|Schnur| Dateiformat (csv/xls/html/mhtml/ods/pdf/xml/txt/tiff/xlsb/xlsm/xlsx/xltm/xltx/xps/png/jpg/gif/emf/bmp/md/Numbers/wmf/svg )|
|Passwort|Schnur||
|istAutoFit|Schnur|wahr falsch|
|onlySaveTable|Schnur|wahr falsch|
|ausPfad|Schnur|neue Dateiposition.|
|Ordner|Schnur|Ursprünglicher Arbeitsmappenordner.|
|Speichername|Schnur|Speichername.|



## REST API

|**API**|**Typ**|**Beschreibung**|**Swagger-Link**|
|:- |:- |:- |:- |
|/Zellen/{Name}|ERHALTEN|Exportiert die Arbeitsmappe in ein anderes Format.|[GetWorkBook](https://apireference.aspose.cloud/cells/#/Workbook/GetWorkBook)|


 Der[OpenAPI-Spezifikation](https://apireference.aspose.cloud/cells/#/Workbook/GetWorkBook) definiert eine öffentlich zugängliche Programmierschnittstelle und lässt Sie REST-Interaktionen direkt von einem Webbrowser ausführen.

 Sie können verwenden**cURL** Befehlszeilentool für den einfachen Zugriff auf Aspose.Cells-Webdienste. Das folgende Beispiel zeigt, wie Sie Cloud API mit cURL anrufen.


{{< tabs tabTotal="1" tabID="11" tabName11="Request" >}}

{{< tab tabNum="11" >}}

```bash

curl -X PUT "https://api.aspose.cloud/v3.0/cells/book1.xlsx?format=pdf" -H "accept: multipart/form-data" -H "Content-Type: multipart/form-data" -H "x-aspose-client: Containerize.Swagger"}
```
