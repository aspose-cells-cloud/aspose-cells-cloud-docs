Dieser REST API gibt `compress`-Daten in einer Excel-Datei an.

- Komprimieren Sie XLS, XLSX, XLSM, XLSB, ODS
- Schnelle Möglichkeit, mehrere Excel-Tabellenkalkulationsdateien zu komprimieren
- Wählen Sie die Komprimierungsstufe
- Unterstützung mehrerer Dateien

## RSET API

```bash

POST https://api.aspose.cloud/v3.0/cells/compress

```

 Die Anforderungsparameter sind:
 
| Parametername| Typ| Pfad/Abfragezeichenfolge/HTTPBody|Beschreibung|
|:- |:- |:- |:- |
| Datei| Datei| Formulardaten| Datei zum Hochladen|
| Komprimierungsstufe| ganze Zahl| Anfrage||
 
 Der[OpenAPI-Spezifikation](https://apireference.aspose.cloud/cells/#/LiteCells/PostCompress) definiert eine öffentlich zugängliche Programmierschnittstelle und lässt Sie REST-Interaktionen direkt von einem Webbrowser ausführen.
 
Sie können das cURL-Befehlszeilentool verwenden, um einfach auf Aspose.Cells-Webdienste zuzugreifen. Das folgende Beispiel zeigt, wie Sie Cloud API mit cURL anrufen.
