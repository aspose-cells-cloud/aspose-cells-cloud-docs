Dieser REST API zeigt an, dass `cells style` auf eine Zelle in einer Excel-Datei gesetzt wird.

 
## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/cells/style
 
```
 Die Anforderungsparameter sind:
 
| Parametername| Typ| Pfad/Abfragezeichenfolge/HTTPBody|Beschreibung|
|:- |:- |:- |:- |
| Name| Schnur| Weg| Name der Arbeitsmappe.|
| Tabellenname| Schnur| Weg| Arbeitsblattname.|
| Bereich| Schnur| Anfrage| Die Reichweite.|
| Stil|| Körper| mit Update-Stileinstellungen.|
| Ordner| Schnur| Anfrage| Der Arbeitsmappenordner.|
| Speichername| Schnur| Anfrage| Speichername.|
 
 Der[OpenAPI-Spezifikation](https://apireference.aspose.cloud/cells/#/Cells/PostUpdateWorksheetRangeStyle) definiert eine öffentlich zugängliche Programmierschnittstelle und lässt Sie REST-Interaktionen direkt von einem Webbrowser ausführen.
 
Sie können das cURL-Befehlszeilentool verwenden, um einfach auf Aspose.Cells-Webdienste zuzugreifen. Das folgende Beispiel zeigt, wie Sie Cloud API mit cURL anrufen.
