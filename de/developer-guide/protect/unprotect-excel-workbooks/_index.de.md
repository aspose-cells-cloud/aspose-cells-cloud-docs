Dieser REST API entschützt einen Excel `workbook`.

**Abfrageparameter**

|Parametername|Typ|Beschreibung|
|:- |:- |:- |
|Ordner|Schnur|Ursprünglicher Arbeitsmappenordner.|
|Speichername|Schnur|Speichername.|

**Body-Parameter anfordern**

|Parametername|Typ|Beschreibung|
|:- |:- |:- |
|Schutz|WorkbookProtectionRequest||

**WorkbookProtectionRequest**

|Parametername|Typ|Beschreibung|
|:- |:- |:- |
|Schutztyp|Schnur|ALLE/INHALTE/KEINE/OBJEKTE/SZENARIEN/STRUKTUR/FENSTER|
|Passwort|Schnur||


## REST API

|**API**|**Typ**|**Beschreibung**|**Swagger-Link**|
|:- |:- |:- |:- |
|/cells/{Name}/Schutz|LÖSCHEN|Heben Sie den Schutz eines Dokuments auf|[DeleteUnProtectDocument](https://apireference.aspose.cloud/cells/#/Workbook/DeleteUnProtectDocument)|

 Der[OpenAPI-Spezifikation](https://apireference.aspose.cloud/cells/#/Workbook/DeleteUnProtectDocument) definiert eine öffentlich zugängliche Programmierschnittstelle und lässt Sie REST-Interaktionen direkt von einem Webbrowser ausführen.

 Sie können verwenden**cURL** Befehlszeilentool für den einfachen Zugriff auf Aspose.Cells-Webdienste. Das folgende Beispiel zeigt, wie Sie Cloud API mit cURL anrufen.

