 Dieser REST API gibt an, einen `digital signature` für eine Excel-Arbeitsmappe hinzuzufügen.
 
## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/{name}/digitalsignature
 
```
 Die Anforderungsparameter sind:
 
| Parametername| Typ| Pfad/Abfragezeichenfolge/HTTPBody|Beschreibung|
|:- |:- |:- |:- |
| Name| Zeichenfolge| Weg| Name der Arbeitsmappe.|
| digitale Signaturdatei| Zeichenfolge| Abfrage| Parameter der digitalen Signaturdatei.|
| Passwort| Zeichenfolge| Abfrage||
| Ordner| Zeichenfolge| Abfrage| Ordner der Arbeitsmappe.|
| Speichername| Zeichenfolge| Abfrage| Speichername.|
 
 Der[OpenAPI-Spezifikation](https://apireference.aspose.cloud/cells/#/Workbook/PostDigitalSignature) definiert eine öffentlich zugängliche Programmierschnittstelle und ermöglicht die Durchführung von REST-Interaktionen direkt über einen Webbrowser.
