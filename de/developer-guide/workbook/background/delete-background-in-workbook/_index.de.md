Dieser REST API gibt an, dass der Hintergrund in einer Excel-Arbeitsmappe gelöscht werden soll.

**Abfrageparameter**

|Parametername|Typ|Beschreibung|
|:- |:- |:- |
|Ordner|Schnur|Ursprünglicher Arbeitsmappenordner.|
|Speichername|Schnur|Speichername.|


## REST API

|**API**|**Typ**|**Beschreibung**|**Ressourcen-Link**|
|:- |:- |:- |:- |
|/cells/{Name}/Hintergrund|LÖSCHEN|Hintergrund in excel datei löschen|[Arbeitsmappenhintergrund löschen](https://apireference.aspose.cloud/cells/#/Workbook/DeleteWorkbookBackground)|

 Der[OpenAPI-Spezifikation](https://apireference.aspose.cloud/cells/#/Workbook/DeleteDecryptDDeleteWorkbookBackgroundocument) definiert eine öffentlich zugängliche Programmierschnittstelle und lässt Sie REST-Interaktionen direkt von einem Webbrowser ausführen.

 Sie können verwenden**cURL** Befehlszeilentool für den einfachen Zugriff auf Aspose.Cells-Webdienste. Das folgende Beispiel zeigt, wie Sie Cloud API mit cURL anrufen.

```java

curl -X DELETE "https://api.aspose.cloud/v3.0/cells/Book1.xlsx/background?folder=DotnetFiles" -H "accept: application/json" -H "x-aspose-client: Containerize.Swagger"

```
