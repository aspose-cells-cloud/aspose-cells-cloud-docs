Dieses REST API kopiert `columns` in ein Excel Arbeitsblatt.

## RSET API
 
```bash
 
POST http://api.aspose.cloud/v3.0/cells/{name}/worksheets/{sheetName}/cells/columns/copy
 
```
 Die Anforderungsparameter sind:
 
| Parametername| Typ| Pfad/Abfragezeichenfolge/HTTPBody|Beschreibung|
|:- |:- |:- |:- |
| Name| Zeichenfolge| Weg|Der Name der Arbeitsmappe.|
| Blattname| Zeichenfolge| Weg| Der Arbeitsblattname.|
| sourceColumnIndex| ganze Zahl| Abfrage| Quellspaltenindex|
|DestinationColumnIndex| ganze Zahl| Abfrage| Index der Zielspalte|
| Spaltennummer| ganze Zahl| Abfrage| Die kopierte Spaltennummer|
| Arbeitsblatt| Zeichenfolge| Abfrage||
| Ordner| Zeichenfolge| Abfrage| Der Dokumentenordner.|

 Der[OpenAPI-Spezifikation](https://apireference.aspose.cloud/cells/#/Cells/PostCopyWorksheetColumns) definiert eine öffentlich zugängliche Programmierschnittstelle und ermöglicht die Durchführung von REST-Interaktionen direkt über einen Webbrowser.

 Sie können verwenden**cURL** Befehlszeilentool für den einfachen Zugriff auf Aspose.Cells-Webdienste. Das folgende Beispiel zeigt, wie Sie die Cloud API mit cURL anrufen.
