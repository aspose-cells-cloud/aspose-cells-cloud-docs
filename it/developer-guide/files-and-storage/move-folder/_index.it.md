## **Excel API: Sposta cartella**

```
PUT http://api.aspose.cloud/v4.0/cells/storage/folder/move/{srcPath}
```

### **Descrizione della funzione**

Questo API consente agli utenti di spostare una cartella da una posizione all'altra all'interno dell'archiviazione cloud Aspose.Cells. È essenziale per organizzare i file e gestire efficacemente l'archiviazione cloud.

###  I parametri di richiesta di**sposta cartella** API sono

| Nome del parametro| Tipo| Percorso/Stringa di query/Corpo HTTP| Descrizione|
|----------------|--|------------------------|--------------------------------------------------------|
| Percorso src| Corda| Sentiero| Percorso di origine della cartella da spostare.|
| Percorso di destinazione| Corda| Domanda| Percorso di destinazione in cui spostare la cartella.|
| NomeArchiviazioneSorgente| Corda| Domanda| Nome dell'archivio di origine.|
| NomeArchiviazioneDest| Corda| Domanda| Nome dell'archivio di destinazione.|

### **Descrizione della risposta**

```json
{
Void
}
```
